Datos abiertos

Fuentes de datos abiertos nacionales

  • Datos.gob.es: Portal de datos abiertos del gobierno de España.
  • INE: Instituto Nacional de Estadística, fuente de estadísticas oficiales en España.

Fuentes de datos abiertos autonómicos

  • Andalucía Open Data: Portal de datos abiertos de la Junta de Andalucía.
  • Ayuntamiento de Sevilla: Portal de datos abiertos del Ayuntamiento de Sevilla.
  • Ayuntamiento de Málaga: Portal de datos abiertos del Ayuntamiento de Málaga.
  • Ayuntamiento de Córdoba: Portal de datos abiertos del Ayuntamiento de Córdoba.
  • Diputación de Cádiz: Portal de datos abiertos de Cádiz.

Fuentes de datos abiertos municipales

  • Ayuntamiento de Madrid: Portal de datos abiertos del Ayuntamiento de Madrid.
  • Open Data BCN: Portal de datos abiertos del Ayuntamiento de Barcelona.

Existen muchas fuentes de datos abiertos disponibles en Internet que se pueden utilizar para el análisis de datos. Algunas de las más populares incluyen:

  • Kaggle: Kaggle es una plataforma de análisis de datos que ofrece una gran cantidad de datos abiertos y retos de análisis de datos.
  • Google Dataset Search: Google Dataset Search es un motor de búsqueda de datos abiertos que permite encontrar conjuntos de datos de una amplia variedad de fuentes.
  • UCI Machine Learning Repository: El UCI Machine Learning Repository es una colección de conjuntos de datos utilizados en investigaciones de machine learning y análisis de datos.
  • Data.gov: Data.gov es un portal de datos abiertos del gobierno de Estados Unidos que ofrece acceso a conjuntos de datos de una amplia variedad de agencias gubernamentales.
  • World Bank: El Banco Mundial ofrece acceso a una gran cantidad de datos sobre economía, población y desarrollo a nivel mundial.

Además, muchas empresas y organizaciones ofrecen conjuntos de datos abiertos a través de sus sitios web o a través de programas de datos abiertos.
